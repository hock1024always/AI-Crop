# AI Corp 开发路线图

## 当前版本: v1.0

### 已完成功能 ✅

- [x] Orchestrator 总控服务
- [x] Agent Runtime 运行时
- [x] WebSocket 实时通信
- [x] REST API 接口
- [x] MCP Skill 系统
- [x] NATS 消息队列集成
- [x] LLM 客户端 (DeepSeek/OpenAI)
- [x] 像素风可视化前端
- [x] RAG 向量库和解题助手
- [x] 编译器插件系统 (LLVM/GCC/Go)
- [x] 监控指标采集
- [x] Prometheus 端点

- [ ] 操作审计记录
- [ ] 执行历史回放

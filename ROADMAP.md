# AI Corp 开发路线图

## 当前版本: v1.2

### 已完成功能

- [x] Orchestrator 总控服务
- [x] Agent Runtime 运行时
- [x] WebSocket 实时通信
- [x] REST API 接口
- [x] MCP Skill 系统
- [x] NATS 消息队列集成

- [x] 像素风可视化前端
- [x] RAG 向量库和解题助手
- [x] 编译器插件系统 (LLVM/GCC/Go)
- [x] 监控指标采集 (24+ Prometheus 指标)

- [x] AI 自我迭代与经验共享（MemGPT/Letta 架构）
- [x] 分层记忆系统（短期/长期/反思/技能/共享）
- [x] 向量语义检索（pgvector IVFFlat 索引）
- [x] 知识共享机制（跨 Agent 经验/技能广播）

- [x] PostgreSQL 16 + pgvector 0.8.0 源码编译部署
- [x] IVFFlat 索引自管理（自动创建/重建）
- [x] pgvector 一键部署脚本
- [x] Grafana Dashboard（18 面板）
- [x] gopsutil 系统资源采集
- [x] Docker Compose 一键部署
- [x] 审计日志中间件（HTTP 请求自动记录 + DB 持久化）
- [x] DAG 工作流引擎（拓扑排序 + 并行执行 + 条件分支 + 重试）
- [x] 工作流 REST API + Agent 桥接
- [x] Token 配额管理（分钟/小时/天三级限制）
- [x] JWT 认证中间件（可选认证 + 角色权限检查）
- [x] 速率限制（滑动窗口算法）
- [x] PII 脱敏（手机号/身份证/银行卡/邮箱，输入+输出双向过滤）
- [x] A2A 协议（Agent 间 P2P 通信 + 协作会话管理）
- [x] 输入安全过滤（XSS/SQL 注入检测）

## Phase 1: 安全增强

**目标**: 增强 Agent 沙箱的安全性和可控性

### 1.1 PII 脱敏

- [x] 敏感数据检测 (手机号、身份证、银行卡、邮箱)
- [x] 自动脱敏处理（输入+输出双向过滤）
- [x] 日志脱敏

### 1.2 预算控制

- [x] Token 配额管理（分钟/小时/天三级限制）
- [x] 请求次数限制（滑动窗口）
- [ ] 按项目/用户计费（需扩展 DB Schema）

### 1.3 审计日志

- [x] 操作审计记录（HTTP 中间件自动记录 + DB 持久化）
- [ ] 执行历史回放
- [ ] 异常行为检测
